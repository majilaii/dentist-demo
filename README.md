# 6 Corners Dental Studio Remake

This is a local Next.js rebuild of the 6 Corners Dental Studio marketing site.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint

## Current scope

- App Router setup in `src/app`
- Homepage sections for hero, services, about, new patients, insurance, FAQ, testimonial, and contact
- Reusable site content in `src/data/site.ts`
- Clean starter styles using Tailwind utilities and a single global stylesheet

## Notes

- Replace placeholder gradients and text-only visuals with approved practice photography and brand assets.
- Expand anchor-based homepage sections into route-based pages as the remake grows.
- Review content copy against the live site before launch.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
